### Code changes (2)

**1. Better title hook patterns in idea generation** (`app/services/video_gen.py`)  
Added hook-pattern guidance to the long-form idea generation prompt: comparison frames (X vs Y), rhetorical questions with stakes, number-driven guides, pattern interrupts, urgency hooks. Motivated by the winner titles' shared pattern vs. the generic fallbacks. Will affect all future idea generations.

**2. Auto-retry renders on transient LLM overload** (`app/services/render_loop.py`)  
When a render fails with a transient error (overloaded_error, rate_limit_error, 529, 503), the video is re-queued to APPROVED status (up to 2 retries) using the existing `retry_count` field instead of going to permanent FAILED. This recovers video 158 and future similar failures automatically.
