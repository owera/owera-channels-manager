# Growth agent reports

One committed markdown file per daily run — `YYYY-MM-DD.md` — written by the
autonomous growth agent (see `../daily-agent-playbook.md`). Each report records what
the agent observed (key analytics), what it learned (winning/losing topics + formats
and its hypotheses), what it did (every API action and code change, with ids/links),
and what to watch next.

These are the audit trail for the self-improving loop: skim them to see, day by day,
what the agent changed and why. Every code change it makes is also a normal git commit
on `main`, so nothing here is the only record — but this is the human-readable diary.
