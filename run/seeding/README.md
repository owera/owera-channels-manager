# Daily seeding kits

The growth agent writes one `YYYY-MM-DD.md` here every run (Subscriber Offensive directive #5).
The **operator** pastes the items — the agent never posts externally.

Rules for every item:
- **Value-first, never a bare link.** The post must teach or provoke a real thought on its own;
  the video link is the "go deeper", not the payload. Match each platform's norms and language.
- Feature the day's (or yesterday's) best video per channel — highest avp or best topic fit.
- No astroturfing: post as Rodrigo, say it's your channel when the platform culture expects it
  (HN/Reddit punish stealth self-promo harder than honest self-promo).

```markdown

...

## Comment drafts (adjacent channels)
- <channel/video>: "<draft>"

## Yesterday's seeds — status
- <item>: posted? traction? (views from EXT_URL in traffic_json if visible)
```
