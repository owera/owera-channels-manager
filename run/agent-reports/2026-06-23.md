## 4. App Improvement

### Change 1 — Short-form script closing punch-line (`app/services/engines/worker.py`)

**Problem:** Top retention videos (51–91%) share a pattern: they deliver a clear verdict and end with a quotable, memorable line. The script prompt previously said "answer the question concisely and directly" — no instruction to take a position or crystallise the lesson at the close.

**Change:** Added two instructions to the short script prompt:
- "give the honest verdict or concrete insight — take a clear position, don't hedge"
- "Close with one tight, memorable line that crystallises the lesson in a sentence the viewer will quote"

**Verified by calling `_generate_script` directly on "Your AI Agent Forgets Everything When the API Crashes":**

```
You spent an hour building context with your AI agent, then one API timeout wipes the whole conversation clean.

Here's the brutal truth: your agent never actually remembered anything. It was replaying the entire chat history on every single call... Persist every step, checkpoint your state, and treat the API like the disposable thing it is.

Your agent doesn't have a memory. You do. Build it.
```

Verdict is clear ("brutal truth"). Close is tight and quotable ("Your agent doesn't have a memory. You do. Build it."). Behavioral change confirmed — not just assumed from code reading.
